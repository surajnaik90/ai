# Anthropic Claude - Targeted to Enterprise customers.

## Capabilities

1. Large context window: 200K+ tokens (~500 pages of text)
2. Claude desktop app: Chat. Cowork. Code.
3. Projects: Manage Project knowledge. Provide Instructions.
4. Artifacts
5. Skills

## Effective prompt writing

1. **Set Stage**: Define role & objectives. Provide context if needed.
2. **Define task**: What task you want Claude to perform?
3. **Specify rules**: What rules to follow? Provide examples for rules.

## The 4D framework for AI fluency

**AI Fluency is the ability to collaborate effectively with AI tools.**

1. **Delegation**: What tasks to performed by human and what tasks by AI?
2. **Descirption**: Effective communication with AI
3. **Discernment**: Evaluating the output generated by AI.
4. **Diligence**: Using AI responsibly and ethically.

## Projects

1. Claude allows you to create projects. Manage knowledge base for context.
   ChatGPT also has this capability.
2. If context exceeds, **RAG mode is enabled**.

## Artifacts [Intention: Easy to access the output generated by Claude]

1. Claude creates artifacts for the query we ask if it meets certain criteria.
   Ex: Documents, images, flowcharts, code snippets etc.
2. If Claude is not creating it for whatever reason, just ask in the prompt itself.
3. Without artifacts, things would have been burried in the chat. Difficult to access. 
   So, artifacts was created.

## Claude's reach: Connectors, Enterprise search and Research mode

1. Claude allows us to do an enterprise search for data present in various tools and data sources.   
   User will be able to access data only if he/she has permissions.
   Ex: Microsoft 365, Gmail, Notion etc are supported.
2. Data is safe. Conversations remain private. It is not used for indexing.
3. **Research mode**: Analyzes data from multiple sources and creates comprehensive detailed research report.   
   It takes around 5 - 15 Minutes. Might 45 minutes for complext tasks.
